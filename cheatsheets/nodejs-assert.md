---
title: assert
category: Node.js
intro: |
  The `assert` module provides a set of assertion functions for verifying invariants.
---

### Assertions

```js
assert(val)
assert.equal(actual, expected)
assert.notEqual(a, e)
```

```js
assert.deepEqual(a, e)
assert.notDeepEqual(a, e)
```

```js
assert.throws(fn)
```
